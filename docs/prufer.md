# **Prüfer コード**

$n$ 頂点ラベル付き木の個数は $n^{n-2}$ である。これは以下のように示される。

### 証明

根付き木をカウントすることにする。連結成分が $k$ 個存在し全ての連結成分が根付き木を為していると仮定する。ここに有向辺を $1$ 本追加し、追加後も全ての連結成分が根付き木であるようにする。有向辺の「根本」は任意の頂点、「先端」は根元の属するもの以外の根に限られることから、このような操作の通り数は $\sum_{i=1}^k (k-1)v_i = n(k-1)$ 通り考えられる。

$k=n,n-1, \cdots, 2$ まで操作数を掛け合わせると $n^{n-1}(n-1)!$ となる。ここで、辺を追加した順番について $(n-1)!$ 個ずつの重複が存在するため、そのような根付き木の個数は $n^{n-1}$ となる。どの頂点を根にするかで $n$ 通りずつ存在するため、無向な木の個数は $n^{n-2}$と求められる。

数えられる→何かの全単射が存在していたらいいな～～→実は木と数列の間に全単射あります

## 定義

頂点 $n$ を根とするラベル付き木に対し、以下の操作で生成される数列 $(y_1, \cdots, y_{n-2})$ を **Prüfer コード** という。

- $k=1,2, \cdots, n-2$ について、葉ノードのうち番号が最小のものを選びその親を $y_k$ とする。その後、選んだ葉を木から削除する。
- 最後に2頂点が残るので、その親として $y_{n-1}=n$ とすることもある。

全単射なので復元もできる。 $k=1, \cdots, n-1$ の順に以下を行う。

- $S=\{x_1, \cdots, x_{k-1}\} \cup \{y_k, \cdots, y_{n-1} \}$ とする。 $x_k=\mathrm{mex}(S)$ とし、 $y_k$ の子を $x_k$ とする。
- $x_k$ は既に復活した頂点、 $y_k$ は現時点で子を持つので削除され得ない頂点だと考え、 $k$ 回目に削除された頂点はそれらを満たさない中で番号が最小の頂点

## 性質

- 頂点 $i$ の次数が $d_i$ であるとき、 $d_i$ は Prüfer コードに $d_i-1$ 回登場する。
